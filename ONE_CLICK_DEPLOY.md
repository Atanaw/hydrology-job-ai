# One-click deployment checklist

## GitHub Actions deployment

1. Upload repository contents to GitHub.
2. Add secrets in GitHub Actions.
3. Open Actions.
4. Run `Daily Hydrology Job Search`.
5. The same workflow runs automatically every day.

## Local Docker deployment

```bash
docker compose up --build
```

## Free cloud option

Use GitHub Actions first. It is free and does not require your laptop to stay on.
