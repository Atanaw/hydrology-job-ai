# GitHub Setup

1. Create a new repository on GitHub.
2. Extract this ZIP file.
3. Upload all extracted files and folders to the repository root.
4. Confirm this file exists: `.github/workflows/daily-job-search.yml`.
5. Go to `Settings → Secrets and variables → Actions`.
6. Add secrets listed in README.md.
7. Go to `Actions → Daily Hydrology Job Search → Run workflow`.

If the workflow does not appear, the `.github/workflows/daily-job-search.yml` file is missing or uploaded inside an extra folder.
